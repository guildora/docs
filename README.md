# Guildora Docs

> **Status: Experimental / Active Development**
> Guildora is under active development. APIs, database schema, and features may change without notice.

Central documentation for Guildora.

Guildora is a Discord-centered community platform consisting of a **Hub** (member web app + landing page editor), **Bot** (Discord event processor), **Marketplace** (community extensions), and an **App Extension System** for community-built plugins.
