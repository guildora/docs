# Guildora Docs — Agent Instructions

## Confidentiality: Marketplace

The Guildora Marketplace is a **private, closed-source** service.
Its repository, implementation details, tech stack internals, API routes, database schema, and architecture must **never** be exposed in public-facing documentation.

**Allowed:** Mention the Marketplace as a product/website (e.g. link to `guildora.app`), describe user-facing workflows (browsing apps, submitting apps, developer portal features).

**Not allowed:** GitHub repo links to the marketplace repo, internal API details, database schema, deployment configuration, code snippets from the marketplace codebase, or any information that implies the source code is available.

The `architecture-systems/marketplace/` directory is an **internal reference** for agents working on the marketplace itself. It must not be copied into public-facing docs, READMEs, or any output visible to end users.

This rule applies to all generated content in this repository.
